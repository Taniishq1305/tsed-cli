    Why Ts.ED?

    Ts.ED is a Node.js Framework on top of Express/Koa.js. Written in Typescript, it helps you build your server-side application easily and quickly. If you want to start a complete out-of-the-box project or fully customize it yourself, Ts.ED will guide you there !

    See more on Ts.ED

    tsed init

    The Ts.ED CLI makes it easy to create an application that already works, right out of the box. It already follows our best practices!

    tsed generate

    Generate provider, controllers, services and pipes with a simple command. The CLI will also create simple test shells for all of these.

    tsed add

    Install a CLI plugins to extend the CLI capabilities. Out-of-the-box support tslint, prettier, mocha, jest, passport, etc...
